## Running Locally

This application requires Node.js v16.13+.

```bash
git clone https://github.com/tconroy/conroy.codes.git
cd conroy.codes
pnpm install
pnpm dev
```

Create a `.env` file similar to [`.env.example`](https://github.com/tconroy/conroy.codes/blob/main/.env.example).
